Welcome to the site of Honku!

Webpage of Honku, a chess engine written in python. This bot was coded by Claude AI, deployed on Lichess.

Current Version: Honku v2

Features of Honku

Lichess Integrated

Honku is fully integrated with Lichess, allowing seamless play on the platform. Play against Honku on Lichess @HonkuWeak!

Python-Based

Honku is fully coded in Python, making the code simpler and easier to understand.

UCI-Ready

If you want to play against Honku in a UCI-compatible chess GUI, you can use the UCI interface that Honku supports.

Technical Features

  • Negamax + Alpha-Beta Pruning
  • Iterative Deepening
  • Null-Move Pruning with Late Move Reduction
  • Quiescence Search
  • Piece Square Tables
  • Other Strategic Evaluation Features
  • Time Management
